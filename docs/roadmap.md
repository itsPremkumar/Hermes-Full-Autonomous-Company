# Long-Term Evolution Roadmap

The project progresses through explicit stages. Each stage is "done" only when its
capability is real and documented, not when it is merely described.

```
V1  Single Agent ............... DONE — Hermes runs the company manually via prompts
        ↓
V2  Multi-Agent ................ DONE — Paperclip 7-agent org (CEO/CFO/COO/CMO/PM/QA/Eng)
        ↓
V3  Memory System .............. DONE — GitHub + Hermes skills + knowledge-base/
        ↓
V4  Business Automation ........ IN PROGRESS — income engine, 8 products, autonomy loop,
                                  Gumroad publish BLOCKED on human (PRE-52)
        ↓
V5  Self-Optimization .......... NEXT — benchmarks.md trends drive auto-improvement;
                                  experiment framework live
        ↓
V6  Distributed AI Company ..... (future) — multiple companies/agents on one kernel
        ↓
V7  Multi-Company OS ........... (future) — kernel manages N companies, cross-company learning
```

## Principle
Incremental build. Keep the core model-agnostic so any actor can be swapped as models
improve. Do not skip to V6/V7 until V4–V5 are provably working with real revenue signal.
Track progress in CHANGELOG.md and this file.

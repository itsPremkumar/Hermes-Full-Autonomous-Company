## Script Format

Each script follows this structure:
```
## [Niche] Script #[Number]: [Title]

[Opening hook — first 3 seconds]

[Body — main content with [Visual: ...] cues]

[Call to action — final 5 seconds]

Avg time: ~45-60s | Tone: [Educational / Inspiring / Shocking / Curious]
```
